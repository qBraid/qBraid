8Overview 

9--------- 

10The qbraid circuit layer is designed as a lightweight infrastructure for creating 

11circuits. Without the machinery to interface with devices directly, the qBraid circuit 

12layer is built primarily for interfacing with the qBraid transpiler. For this reason, 

13the structure of this circuit layer mirrors that of the transpiler 

14wrapper classes closely. 

15 

16The circuit layer also mirrors cirq in its use of moments and instructions. However, 

17parameters are stored globally. Qubits are simply integers. 

18 

19Example Usage 

20-------------- 

21 

22To create a circuit simply, the following code will suffice. 

23 

24 .. code-block:: python 

25 

26 import numpy as np 

27 from qbraid.circuits import Circuit, RX, drawer 

28 

29 circuit = Circuit(3) #create a circuit with 3 qubits 

30 circuit.add_instruction('H',0) 

31 circuit.add_instruction('RX',np.pi/2,2) 

32 circuit.add_instruction('CX',[1,2]) 

33 

34 crx_gate = RX(np.pi/4).control() 

35 circuit.append(crx_gate([0,1])) 

36 

37 drawer(circuit) 

38 ... 

39 

40The string passed to the ``add_instruction`` argument follows the convention of the 

41transpiler, which is documented here. 

42 

43For more explicit control over Moments, Instructions, and Gates, the following code shows 

44how circuits can be created more directly. 

45 

46 .. code-block:: python 

47 

48 from qbraid.circuits import Circuit, Instruction, Moment, H, RX, drawer 

49 from numpy import pi 

50 

51 circuit = Circuit(3) 

52 

53 h = H() 

54 instr_h0 = Instruction(h,0) 

55 instr_h1 = Instruction(h,1) 

56 

57 moment = Moment([instr_h0,instr_h1]) 

58 

59 instr_rx0 = RX(pi/2)(0) 

60 instr_rx2 = RX(pi/4)(2) 

61 circuit.append(moment) 

62 circuit.append(instr_rx0) 

63 circuit.append(instr_rx2) 

64 

65 drawer(circuit)
